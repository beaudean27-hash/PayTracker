# PayTracker - Work & Payment Tracker

A React-based web application to track work days and payments.

## Features

- ✅ Add work days (full day or half day)
- ✅ Mark days as paid with confirmation
- ✅ Edit work day dates and types
- ✅ Delete work days with multi-step confirmation for paid days
- ✅ Three tabs: Unpaid Days, Payment History, and Deleted Days
- ✅ Restore deleted days
- ✅ Statistics dashboard showing unpaid, paid, and total days
- ✅ Data persistence using localStorage

## Getting Started

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

The app will be available at http://localhost:3000

### Build for Production
```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)
- localStorage for data persistence
